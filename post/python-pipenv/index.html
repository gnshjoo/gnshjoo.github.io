Python Pipenv 가상환경 셋팅하기

October 8, 2020 , posted under python Django

1. Python 가상환경 설정

Python3이 설치 되어있다는 가정하에 설명하도록 하겠다.

$ pip3 install pipenv
$ pipenv --version
pipenv, version 2020.6.2

pipenv을 사용하려면 디렉토리를 만들어 준 후 디렉토리로 이동한다.

$ mkdir django-test && cd django-test
$ pipenv shell
